# ST 554 Project 2

This repository documents my work for Project 2. The project consists of two parts:

- In Part I, I construct a custom class with methods that streamline the data validation process for `pyspark.sql` dataframes. The custom class' source code is in **SparkDataCheck.py**, and its functionality is demonstrated in **Project 2 Notebook.ipynb**.
- In Part II, I explore the relative strengths and weaknesses of `pyspark.sql` and `pyspark.pandas` for data manipulations. This exploration is documented in **Project 2 Notebook.ipynb**, where I complete the same set of data manipulations using each method to formally compare and contrast their functionality and capabilities.
